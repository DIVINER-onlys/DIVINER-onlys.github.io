从零开始使用webpack构建vuejs开发项目

Posted on 2018-05-01 | In Vue2.0开发过程中遇到的坑   |   次阅读

概要

很多人在vue的官网学习完之后，往往要开始构建第一个项目时不知道到从何下手，本文的目的就是带领你一步一步使用webpack工具构建出完整的vuejs开发项目的目录

前提

  • 你已经了解vue的基础知识，尝试过使用vue-cli官方脚手架搭建项目。

安装

要使用npm命令就需要先安装node.js, npm是随node.js安装的，所以需要先安装node.js，因为node.js安装比较容易就不在这里演示，
这里给个传送门：node.js安装下载地址：http://nodejs.cn/download/

一般，我们还会安装淘宝镜像cnpm，因为在墙内，有时候使用npm安装会很慢，所以需要。

vue脚手架的安装

上述命令的就是使用npm安装vue-cli这个脚手架，-g就是全局安装，安装完毕就可以使用vue命令了
我们接下来就使用它的命令行工具 vue-cli 来构建项目。

项目搭建

Step1.0 项目初始化

建立可以运行的项目（打包工具：webpack）（指令作用介绍如下）

（1）找一个盘中的一个文件夹，在里面新建vue项目
vue init webpack my-project // 初始化项目
（2）进入my-project
cd my-project
（3）把配置文件的插件安装一遍
npm install // 安装 项目依赖
（4）运行开发环境
npm run dev // 在localhost启动测试服务器
（5）把项目编译成可以上线的静态html css js
npm run build // 生成上线目录（部署）
